## Mini-projet Java : Jeu de personnages ##

## Lancer le jeu ##

1. Ouvrir le projet dans IntelliJ IDEA.
2. Dans le module `MiniModule`, ouvrir le fichier `src/Jeu.java`.
3. Cliquer sur le triangle vert à gauche de la méthode `main` puis choisir **Run 'Jeu.main()'**.
4. Saisir dans la console :
   - le nom des deux joueurs ;
   - les noms des 3 personnages de chaque joueur ;
   - le numéro des personnages pour attaquer tour par tour.
5. Le jeu se termine quand un joueur n’a plus de personnages vivants et le vainqueur s’affiche dans la console.
